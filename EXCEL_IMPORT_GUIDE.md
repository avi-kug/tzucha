## הוראות צעד אחר צעד

### 1. פתח Excel חדש
- פתח Microsoft Excel / LibreOffice Calc / Google Sheets

### 2. צור את שורת הכותרות (שורה 1)
העתק את הכותרות הבאות לשורה הראשונה:

```
מזהה | תאריך יצירה | תאריך עדכון | שם עמדה | שם פרטי | שם משפחה | מספר זהות | עיר | רחוב | מס' טל' | מס' נפשות בבית (כולל ההורים) | מס' ילדים נשואים | מקום לימודים / עבודה | סכום הכנסה / מלגה | מקום לימודים / עבודה | סכום הכנסה / מלגה | קצבת ילדים | קצבת שארים | קצבת נכות | הבטחת הכנסה | השלמת הכנסה | סיוע בשכר דירה | מקור הקצבה אחר | סכום | הוצאות דיור | הוצאות שכר לימוד (סכום כולל למוסדות) | הוצאה חריגה קבועה | פירוט - הוצאה חריגה | פרט מה סיבת הקושי | הערות | שם בעל החשבון | בנק | סניף | מס' חשבון | שם מבקש התמיכה | מספר עסקה
```

### 3. הוסף נתונים
מלא את השורות הבאות עם הנתונים. **חובה למלא:**
- שם פרטי (E)
- שם משפחה (F)
- מספר זהות (G)
- מס' נפשות בבית (K)

### 4. טיפים חשובים

#### ✅ מספרי זהות
- **חשוב מאוד!** משמש לזיהוי ושיוך אוטומטי
- ודא שהמספר תקין (9 ספרות)
- אל תשתמש במספר זהות כפול

#### ✅ סכומים
- השתמש במספרים בלבד (ללא ₪)
- דוגמא נכונה: `1500`
- דוגמא שגויה: `1,500 ₪`

#### ✅ תאריכים
- השאר עמודות B ו-C ריקות (אוטומטי)
- אל תמלא את עמודת A (מזהה) לרשומות חדשות

#### ✅ נפשות בבית
- **חובה!** ללא זה החישובים לא יעבדו
- כלול את כל בני הבית כולל ההורים
- דוגמא: 5 (אבא, אמא, 3 ילדים)

### 5. שמור את הקובץ
- פורמט: `.xlsx` (Excel 2007 ומעלה)
- שם מוצע: `supports_import_YYYY-MM-DD.xlsx`

## דוגמאות Excel מוכנות

### דוגמא 1: משפחה עם הכנסה נמוכה
```
שם: דוד לוי
נפשות: 6
הכנסות: 4000 (עבודה) + 1000 (קצבת ילדים) = 5000
הוצאות: 2500 (דיור) + 1000 (לימוד) = 3500
הכנסה לנפש: (5000-3500)/6 = 250
תמיכה צפויה: 250 < 700 → 200 × 6 = 1200 ₪
```

### דוגמא 2: משפחה עם הכנסה בינונית
```
שם: משה כהן
נפשות: 4
הכנסות: 6000 (עבודה) + 600 (קצבת ילדים) = 6600
הוצאות: 3000 (דיור) + 800 (לימוד) = 3800
הכנסה לנפש: (6600-3800)/4 = 700
תמיכה צפויה: 700-800 → משלים עד (900×4) - 6600 = -3000 → עד 900/נפש
```

### דוגמא 3: משפחה עם הכנסה גבוהה
```
שם: יעקב אברהם
נפשות: 3
הכנסות: 8000 (עבודה) + 400 (קצבת ילדים) = 8400
הוצאות: 2000 (דיור) + 500 (לימוד) = 2500
הכנסה לנפש: (8400-2500)/3 = 1966
תמיכה צפויה: 1966 > 800 → 100 × 3 = 300 ₪
```

## נוסחאות Excel מועילות (אופציונלי)

אם אתה רוצה לבדוק את החישובים ב-Excel לפני הייבוא:

### חישוב סה"כ הכנסות (עמודה AK)
```excel
=SUM(N2:P2,Q2:X2)
```

### חישוב סה"כ הוצאות (עמודה AL)  
```excel
=SUM(Y2:Z2,AA2)
```

### חישוב הכנסה לנפש (עמודה AM)
```excel
=(AK2-AL2)/K2
```

### חישוב סכום תמיכה (עמודה AN)
```excel
=IF(AM2<700, 200*K2, IF(AM2<800, (900*K2)-AK2, 100*K2))
```

## שגיאות נפוצות

### ❌ "שגיאה בייבוא"
**סיבות אפשריות:**
- פורמט הקובץ לא `.xlsx`
- שורת הכותרות חסרה
- עמודות לא בסדר הנכון

**פתרון:** השתמש בטמפלייט המוצע

### ❌ "חלוקה באפס"
**סיבה:** מס' נפשות בבית = 0 או ריק

**פתרון:** מלא את עמודה K (מס' נפשות)

### ❌ "מספר זהות כפול"
**סיבה:** אותו מספר זהות כבר קיים

**פתרון:** המערכת תעדכן את הרשומה הקיימת (אם יש שינויים)
